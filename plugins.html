Plugins JavaScript do Bootstrap 5

Os plugins JS do Bootstrap adicionam interatividade e dinamismo aos seus componentes. Veja exemplos práticos de uso e inicialização dos principais recursos.

Dica: Todos os plugins abaixo funcionam automaticamente ao incluir o bootstrap.bundle.min.js no seu HTML.

Modal

O Modal exibe caixas de diálogo sobre o conteúdo. Pode ser aberto via botão ou programaticamente.

Exemplo de Modal
Este é um modal Bootstrap totalmente funcional! 
<button data-bs-toggle="modal" data-bs-target="#exemploModal">Abrir Modal</button>
<div class="modal fade" id="exemploModal">...</div>

Tooltip

Tooltips mostram dicas ao passar o mouse ou focar em um elemento.

<button data-bs-toggle="tooltip" title="Dica!">...</button>
Inicialização: Tooltips precisam ser inicializados via JS: bootstrap.Tooltip
const tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'));
tooltipTriggerList.map(el => new bootstrap.Tooltip(el));

Toast

Toasts exibem notificações rápidas e não intrusivas.

Notificação Agora
Este é um toast Bootstrap! 
<div class="toast" id="exemploToast">...</div>
JS: document.getElementById('mostrarToast').onclick = function() {
  new bootstrap.Toast(document.getElementById('exemploToast')).show();
}

Offcanvas

O Offcanvas exibe um painel lateral, útil para menus ou informações extras.

Menu Lateral
Conteúdo do painel lateral (offcanvas). 
<button data-bs-toggle="offcanvas" data-bs-target="#exemploOffcanvas">...</button>
<div class="offcanvas" id="exemploOffcanvas">...</div>

Collapse

O Collapse permite expandir/ocultar conteúdo facilmente.

Mostrar/Esconder conteúdo

Este conteúdo pode ser expandido ou recolhido. 
<a data-bs-toggle="collapse" href="#exemploCollapse">...</a>
<div class="collapse" id="exemploCollapse">...</div>

Carousel

O Carousel exibe slides de imagens ou conteúdo.

<div class="carousel slide" id="exemploCarousel">...</div>
